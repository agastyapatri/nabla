# **NABLA**
_Solving Partial Differntial Equations using Neural Networks_

PDEBench is a benchmark suite, providing datasets of physical systems dictated by PDEs. Many efforts have been made to solve PDEs using Deep Learning. This repository is an attempt to implement those methods, using their performance on PDEBench as a method of comparing them. 

The _raison d'etre_ of this project is to serve as a library that provides all these methods with a homogeneous API.
